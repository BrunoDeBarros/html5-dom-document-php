# IvoPetkov\HTML5DOMDocument::modify

- HTML5DOMDocument::FIX_DUPLICATE_METATAGS,
- HTML5DOMDocument::FIX_MULTIPLE_HEADS,
- HTML5DOMDocument::FIX_MULTIPLE_BODIES,
- HTML5DOMDocument::OPTIMIZE_HEAD
